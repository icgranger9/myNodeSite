down4sound

down4sound is a project that I began at Fordham's local hack day

The original goal of down4sound was to create a virtual piano keyboard, using the leap motion sensor. This was not possible in the alloted time perion, so the focus of down4sound shifted to something like virtual bongos, where the user swipes with their left hand to produce one note, and with their right hand to produce a higher note. With this funcionallity, the user is able to produce a rudamentary beat.

While the original goal was never reached, I am still verry happy with this project, and I placed second in fordham's local hack day because of it.
